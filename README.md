# Ejercicio Práctico - Manejo de Promesas en JavaScript

## Descripción
Este ejercicio demuestra el uso de promesas en JavaScript para manejar operaciones asincrónicas, validaciones y procesos pesados.

### Archivos incluidos
- `index.html`: Página principal para ejecutar el ejercicio en el navegador.
- `promesas_ejercicio.js`: Código JavaScript con las tres promesas solicitadas.

## Instrucciones de uso
1. Abre `index.html` en tu navegador.
2. Abre la consola de desarrollador (F12 o clic derecho → Inspeccionar → Consola).
3. Observa los mensajes y resultados de las promesas en la consola.

## Ejercicios implementados
1. **Promesa con setTimeout:** Simula una consulta a base de datos y muestra mensajes en consola.
2. **Promesa según segundos actuales:** Resuelve o rechaza según los segundos y su paridad.
3. **Promesa con proceso pesado:** Ejecuta un bucle grande y explica el delay observado.
